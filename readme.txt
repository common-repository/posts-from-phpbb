== Description ==

New Posts from phpBB is a widget plugin that grabs your recent phpBB forum posts for you to display on your WordPress site 

= Features =

- Connects WordPress with your phpBB forum database 
- Allows you to select the number of recent forum posts to be displayed on your Wordpress site
- Works with the official Classic Widgets plugin, and also with the block-based widget editor
- Available in English (default) and German. You can add your own translation if you want

== Installation ==

* Go to your WordPress Dashboard
* Click on Plugins -> Add New
* Search for "**New Posts from phpBB**"
* Find the plugin and click on the Install Now button
* After installation, click on Activate Plugin link to activate the plugin
* Go to Appearance, select the "New Posts from phpBB" widget and drag it to the area you wish to display it in, then fill in the required fields, done!
